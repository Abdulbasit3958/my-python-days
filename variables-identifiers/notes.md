# Variables

- Variable is just a name for storing value in memory.
- Example: rectangle side = a and b  
  - area = a * b  
  - perimeter = 2 * (a + b)

- Square has both sides same so area = side * side.

- Variables can store numbers, text, or True/False.

---

## Examples
```python
name = "abdul"
age = 21
print(name, age)

name = "rahim"
age = 40
print(name, age)

product = "mango"
weight = 4.5
print(product, weight)

is_adult = True
print(is_adult)
